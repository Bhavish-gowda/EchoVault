# 🔮 EchoVault

### Your Personal AI Knowledge Vault

EchoVault is a local AI-powered knowledge assistant built with C++, vector search, and Retrieval-Augmented Generation (RAG).

Store documents, generate embeddings, perform semantic search, and ask questions using local AI models — all without relying on cloud APIs.

---

## ✨ Features

- 🧠 Local AI Assistant powered by Ollama
- 🔍 Semantic Search using Vector Embeddings
- ⚡ HNSW, KD-Tree and Brute Force Search Algorithms
- 📄 Document Embedding and Storage
- 🤖 Retrieval-Augmented Generation (RAG)
- 📊 Interactive Vector Space Visualization
- 🌐 Built-in REST API
- 🔒 Fully Local — No External AI APIs Required

---

## 🚀 What EchoVault Does

Traditional search relies on keywords.

EchoVault understands the meaning behind your text.

### Workflow

```text
Document
    ↓
Embedding Generation
    ↓
Vector Database
    ↓
Semantic Search
    ↓
Context Retrieval
    ↓
Local LLM
    ↓
AI Response
```

This enables users to upload knowledge, search intelligently, and receive context-aware answers from a local AI model.

## 🎯 Why EchoVault

Most AI applications rely entirely on external APIs.

EchoVault demonstrates how modern AI systems such as ChatGPT Memory, Perplexity, Pinecone, Chroma, and Weaviate work internally by combining:

- Vector Databases
- Embeddings
- Semantic Search
- Local LLMs
- RAG Pipelines

All running locally on your machine.
